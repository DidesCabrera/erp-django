# Nutrition Logic

Storage:
- Macros stored in grams only
- Calories are always derived, never stored

Energy factors:
- Protein: 4 kcal / g
- Carbs: 4 kcal / g
- Fat: 9 kcal / g

Allocation (alloc):
- Percentage of total calories per macro
- Computed after aggregation
- Never averaged

Alloc at higher levels is emergent, not inherited.
