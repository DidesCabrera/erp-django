# Nutrition Platform – Project Overview

This project is a Django-based nutrition planning platform designed for serious,
structured nutrition use cases (bodybuilding, performance, health optimization).

The platform allows users to work across four hierarchical levels:

- Food: atomic nutritional unit
- Meal: composition of foods with quantities
- DailyPlan: scheduled meals across a day
- Program: multiple daily plans across a timeline

The core value of the platform lies in aggregation:
nutrition information becomes meaningful as it scales upward.
